# Master AI Forensic Intelligence System

A powerful AI-driven forensic analysis platform built with Google Gemini AI Studio and modern web technologies.

## Features

- **AI-Powered Analysis**: Leverage Google Gemini for intelligent forensic data processing
- **Real-time Processing**: Fast analysis and insights on forensic evidence
- **User-Friendly Interface**: Intuitive UI for easy navigation and data visualization
- **Secure Data Handling**: Enterprise-grade security for sensitive forensic information

## Tech Stack

- **Frontend**: React + TypeScript
- **UI Framework**: Tailwind CSS
- **AI Engine**: Google Gemini API
- **Build Tool**: Vite
- **Package Manager**: npm

## Prerequisites

- Node.js (v16 or higher)
- npm or yarn
- Google Gemini API key
